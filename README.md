# SudokuGA - Resolvedor de Sudoku con Algoritmo Genético

Proyecto Java orientado a objetos para resolver Sudoku mediante algoritmos genéticos.

## Estructura del Proyecto

```
SudokuGA/
src/
    Main.java
    model/
        Sudoku.java
        Cromosoma.java
    ga/
        AlgoritmoGenetico.java
        Fitness.java
        Seleccion.java
        Cruza.java
        Mutacion.java
        Poblacion.java
    io/
        LectorSudoku.java
    gui/
        VentanaPrincipal.java
        PanelSudoku.java
sudokus/
    facil.txt
    medio.txt
    dificil.txt
```

## Características

- Resolución de Sudoku mediante algoritmo genético
- Interfaz gráfica con Swing
- Lectura de archivos de Sudoku
- Visualización de soluciones
- Soporte para múltiples niveles de dificultad
